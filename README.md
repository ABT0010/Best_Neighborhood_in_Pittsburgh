We are using datas obtained from WPRDC to find the best neighborhood in Pittsburgh.

# Datas used by Abel Sebhatu
- For Number of POGOH stations → https://data.wprdc.org/dataset/station-locations
- For Public School locations → "https://data.wprdc.org/dataset/pittsburgh-public-school-locations"
- For Areas of Respective Neighborhoods used → "https://data.wprdc.org/dataset/neighborhoods2/resource/4af8e160-57e9-4ebf-a501-76ca1b42fc99"
# Datas used by Sharon Joseph
- For Number of Housing Loans Distributed → "https://data.wprdc.org/dataset/aced-allegheny-home-improvement-loan-program-ahilp"

### According to the Burgh Explorers, "The best neighborhoods in Pittsburgh are those that offer both convenient mobility (bike stations & docks) and strong educational resources (number of public schools)." 
### This repository is aimed to find the best neighborhood in Pittsburgh in terms of neighborhood’s overall accessibility, infrastructure quality, and support for daily life.
